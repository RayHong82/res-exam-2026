## 考试格式 / Exam format
每卷 **80 题 = Section A 60 题（各1分）+ Section B 案例 20 题（各2分）= 100 分**，**2.5 小时/卷**，**及格 60%**，两卷独立通过。

## 怎么用 / How to use
从 `00-README.md` 的 6 周计划开始；刷题时先盖住 `> ✅` 答案块自测，再展开核对。建议用 **Obsidian**（打开本文件夹为 vault）或 **VS Code**（`Cmd+K V` 侧边预览）阅读。
