3. 建议 20:00–22:00 发布 → 发布

1. 按 `tiktok-post-1.md` 脚本拍/剪 35秒内竖屏视频（CapCut自动字幕）
2. caption整段复制、选5–8个标签、配trending sound
3. 建议 SG晚8–10点发布 → 发布
